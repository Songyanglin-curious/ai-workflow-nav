# 技术选型

## 主语言与运行时

- 主语言：`TypeScript`
- 服务运行时：`Node.js`
- 包管理：`pnpm`

## 前端技术栈

- 前端框架：`Vue 3`
- 构建工具：`Vite`
- 路由：`Vue Router`
- 状态管理：`Pinia`
- 树与节点可视化：`@vue-flow/core`
- ui组件库：`Naive UI`

> 我熟悉vue并且对于flow的要求不高，所以选定本技术栈而非 react 及其flow

## 4.3 本地服务技术栈

- Web 框架：`Fastify`
- 请求与响应校验：`Zod`
- 配置校验：`Zod`
- 本地配置文件：`JSONC`
- 数据访问风格：`轻量 SQL + 少量仓储封装`

> 够用且格式清晰。
