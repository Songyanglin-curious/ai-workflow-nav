# ai-workflow-nav

Personal AI Workflow Navigator-个人 AI 工作流导航器

它不试图替人思考，也不追求自动完成复杂任务；它只解决一个更真实的问题：在日常使用 AI 时，人很容易在对话中不断开分支，提示词、记录、总结和当前进度分散在各处，导致上下文难以续接、脑容量占用过高、回到任务时恢复成本很大。

本项目的目标，是把这些原本需要人脑临时维持的内容外置出来，形成一个可定位、可切换、可续接的工作界面。核心聚焦两件事：提示词的组织，问题的组织。

提示词用于承载可复用的执行内容；问题树用于承载当前处理中的问题、分支和状态。节点由人维护，节点可以关联提示词、工作流、对话记录和总结入口，用来表达“我现在在哪”“这个分支在处理什么”“下一步该调用什么”。

系统采用“前端壳层 + 本地服务 + 文件层”的结构。所有业务数据的合法读写入口都是本地服务；文件内容是最终事实源；SQLite 只承担索引、关系和查询能力。同步只允许手动触发，不做自动覆盖。

个人 AI 工作流导航器 的核心价值不是知识管理，而是降低记忆负担、降低任务续接成本，让人与 AI 的协作从“不断迷路”变成“可以持续推进”。
