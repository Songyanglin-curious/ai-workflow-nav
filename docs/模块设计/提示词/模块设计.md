## 功能

* 新建提示词
* 列表浏览
* 按名称 / 类别检索
* 查看正文
* 编辑元数据
* 编辑正文
* 复制正文
* 调用 VSCode 打开对应 `.md` 文件
* 删除提示词

## 数据

* 元数据字段：`id`、`name`、`description`、`category`、`content_file_path`、`created_at`、`updated_at`
* `id` 使用 `UUIDv7`
* `content_file_path` 存相对路径
* 正文文件名遵循 `baseName_shortId.md`，`shortId` 为 `id` 后 6 位

## 核心流程

1. 新建：填写 `name`、`description`、`category` → 服务生成 `UUIDv7` → 生成 `baseName_shortId.md` 相对路径 → 创建正文文件 → 写入 `prompts`
2. 读取列表：查 SQLite；名称、类别检索都走元数据查询
3. 读取正文：先查 SQLite 取 `content_file_path`，再按相对路径读文件
4. 编辑元数据：前端显式提交保存 → 服务更新 `name`、`description`、`category`，并更新 `updated_at`
5. 编辑正文：前端显式提交新内容 → 服务写文件 → 更新 `updated_at`
6. 复制：服务读取正文并返回给前端，由前端写入剪贴板
7. 打开文件：前端发起请求 → 服务将 `content_file_path` 解析为绝对路径 → 调用 VSCode 打开
8. 删除：由同一服务顺序完成删库与删文件；任一步失败都直接报错，并提示人工处理残留状态
